## The Problem It Solves

Imagine you want to make a game where the computer "thinks" of a number and the player tries to guess it. Without variables and `if` statements, you couldn't store the secret number or react differently to a right or wrong answer.

With what you know now, you *can* build this:

```python
secret_number = 7
guess = int(input("What is your guess? "))

if guess == secret_number:
    print("You got it!")
if guess < secret_number:
    print("Too low!")
if guess > secret_number:
    print("Too high!")
```

Simple â€” but already a real, working game!

## Understanding the Key Ideas

| Concept | What It Does | Example |
|---|---|---|
| `input()` | Gets text from the user | `name = input("Your name? ")` |
| `int()` | Converts text to a whole number | `guess = int(guess)` |
| `str()` | Converts a number to text | `str(secret_number)` |
| `if` statement | Runs code only when a condition is true | `if guess == secret_number:` |

Think of it this way:
- `input()` always gives you a **string** â€” even if the user types a number
- You need `int()` to turn that string into a number you can compare
- You need `str()` to turn numbers back into text when printing them alongside words

## âš ï¸ Common Mistakes to Avoid

**Forgetting `int()` on the guess**
```python
# âŒ This compares a string to a number â€” will never be equal!
guess = input("What is your guess? ")
if guess == secret_number:

# âœ… Convert first
guess = int(input("What is your guess? "))
```

**Forgetting `str()` when printing the secret number**
```python
# âŒ This will crash â€” you can't join a number and a string with +
print("The secret number was " + secret_number + ".")

# âœ… Wrap it in str()
print("The secret number was " + str(secret_number) + ".")
```

**Using `=` instead of `==` in your if statement**
```python
# âŒ Single = assigns a value, it doesn't compare
if guess = secret_number:

# âœ… Double == checks if they are equal
if guess == secret_number:
```

**Not adding spaces in your greeting string**
```python
# âŒ Produces "Hello,Sarah!"
print("Hello," + name + "!")

# âœ… Include the space inside the quotes
print("Hello, " + name + "!")
```
